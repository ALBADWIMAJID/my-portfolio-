# my-portfolio-

Live site: https://albadwimajid.github.io/my-portfolio-/

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## GitHub Pages

This project uses Vite with `base: "/my-portfolio-/"` in `vite.config.js`.
`public/404.html` redirects to the base path so refreshes on subroutes keep working on GitHub Pages.
If you change the repository name, update `base` and the redirect path.
